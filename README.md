## A remake of the original Keystrokes mod by Canelex.

### Allows customizing:

    Scale
    Mouse Keystrokes
    Jump Keystrokes
    Arrow Keystrokes
    Pressed / Unpressed Background Color
    Pressed / Unpressed Text Color
    Fading Time
    Shadow
    Show in Chat
    Show in GUIs
    Show in F3
